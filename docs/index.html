Before you begin

This is an iOS class. To complete the coding exercises, you will need an Apple development environment with either a simulator or iOS device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

Maps in the palm of our hand has changed the way that we interact with the world. It's hard to imagine navigating without our smartphone these days. In this course, we will examine how to embed and use maps in our iOS applications. Along the way, we will look at how to show the device's current location on the map, change the camera's view, and add pin annotations.

Objectives

  1. Explore iOS mapping options
  2. Use location data
  3. Adjust the viewport of the map
  4. Add annotations to the map

Lecture

IOS230 - Maps and Location.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Add a map and set the properties
    Exercise
    In this exercise, you will display a map using MKMapView in your application and show three different map types.
  2. Show the device's current location
    Exercise
    Next, you will request "in-use" location permissions from iOS and display the device's current location on your map.
  3. Changing the map's viewport with the camera
    Exercise
    Here, you will work with MKMapCamera and change the camera's view for different areas on the map by tapping different buttons.
  4. Add an annotation to your map
    Exercise
    In this final exercise, you will add annotations to a map to display where Xamarin office locations around the world are using MKPointAnnotation.
